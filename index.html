FORLAB 사용법

1. 이미지 업로드

시작 화면에서 이미지를 드래그 앤 드롭하거나 클릭하여 파일을 선택합니다. 여러 장을 한 번에 업로드할 수 있습니다.

2. 감지 모드 선택

툴바의 모드 버튼으로 원형(페트리 접시) 또는 사각형(젤) 감지 모드를 전환합니다.

3. 자동 감지 및 수동 조정

이미지가 로드되면 자동으로 영역이 감지됩니다. 감지된 영역을 드래그하여 위치를 조정하거나, 가장자리를 드래그하여 크기를 변경할 수 있습니다.

4. 크롭 크기 조절

툴바의 크롭 슬라이더로 크롭 영역의 크기를 60%~120%까지 조절합니다.

5. 확인 및 크롭

확인 버튼으로 각 이미지를 확정하거나, 전체 자동 확인으로 모든 이미지를 한 번에 확정합니다. 확정 후 전체 크롭을 누르면 결과가 우측 패널에 표시됩니다.

6. 결과 다운로드

크롭된 이미지를 개별 또는 전체 다운로드할 수 있습니다. 클립보드 복사와 공유 기능도 지원합니다.

FORLAB - 실험실 플레이트 자동 크롭 도구

FORLAB이란?

FORLAB은 실험실에서 촬영한 페트리 접시(Petri dish)와 젤 전기영동(Gel electrophoresis) 사진을 자동으로 감지하고 크롭하는 무료 온라인 도구입니다. 연구원, 대학원생, 실험실 기술자가 실험 결과 사진을 정리할 때 반복적으로 수행하는 이미지 크롭 작업을 자동화하여 시간을 절약해 줍니다.

브라우저에서 바로 실행되며, 이미지가 서버로 전송되지 않아 개인정보와 실험 데이터가 안전하게 보호됩니다. 모든 이미지 처리는 사용자의 컴퓨터에서 직접 수행됩니다.

주요 기능

  • 원형 감지 (페트리 접시) - Hough 변환 알고리즘을 사용하여 원형 페트리 접시를 자동으로 감지합니다. 한 장의 사진에 여러 개의 접시가 있어도 각각을 개별적으로 인식합니다.
  • 사각형 감지 (젤 전기영동) - 색상/채도 기반, 명암 기반, 에지 기반 등 다양한 감지 방법을 조합하여 젤 전기영동 사진의 영역을 정확하게 감지합니다.
  • 수동 조정 - 자동 감지 결과가 정확하지 않을 경우, 드래그하여 위치와 크기를 수동으로 조정할 수 있습니다.
  • 일괄 처리 - 여러 장의 사진을 한 번에 업로드하고, 전체 자동 확인 및 전체 크롭 기능으로 빠르게 처리할 수 있습니다.
  • 크롭 크기 조절 - 슬라이더를 통해 크롭 영역의 크기를 60%에서 120%까지 세밀하게 조절할 수 있습니다.
  • 다양한 내보내기 - 개별 다운로드, 전체 ZIP 다운로드, 클립보드 복사, 웹 공유 API를 지원합니다.

사용 방법

1

이미지 업로드

시작 화면에서 이미지를 드래그 앤 드롭하거나 클릭하여 파일을 선택합니다. JPEG, PNG 등 일반적인 이미지 형식을 지원하며, 여러 장을 한 번에 업로드할 수 있습니다.

2

감지 모드 선택

툴바의 모드 버튼으로 원형(페트리 접시) 또는 사각형(젤) 감지 모드를 전환합니다. 사진의 유형에 맞는 모드를 선택하세요.

3

자동 감지 확인 및 조정

이미지가 로드되면 자동으로 플레이트 영역이 감지됩니다. 감지된 영역의 위치나 크기가 정확하지 않다면 마우스로 드래그하여 조정할 수 있습니다.

4

확인 및 크롭

"확인" 버튼으로 각 이미지를 확정하거나, "전체 자동 확인"으로 모든 이미지를 한 번에 확정합니다. 확정 후 "전체 크롭"을 누르면 우측 패널에 크롭된 결과가 표시됩니다.

5

다운로드

크롭된 이미지를 개별 다운로드하거나, "전체 다운로드" 버튼으로 ZIP 파일로 한 번에 받을 수 있습니다.

자주 묻는 질문 (FAQ)

이미지가 서버로 전송되나요?

아닙니다. FORLAB은 100% 브라우저 기반으로 동작하며, 모든 이미지 처리는 사용자의 컴퓨터에서 직접 수행됩니다. 이미지가 외부 서버로 전송되지 않으므로 실험 데이터의 보안이 보장됩니다.

어떤 이미지 형식을 지원하나요?

JPEG, PNG, WebP, BMP 등 브라우저에서 지원하는 모든 이미지 형식을 사용할 수 있습니다. 고해상도 이미지도 처리 가능합니다.

감지가 정확하지 않으면 어떻게 하나요?

자동 감지 결과가 만족스럽지 않을 경우, 감지된 영역을 마우스로 드래그하여 위치를 이동하거나 가장자리를 드래그하여 크기를 조정할 수 있습니다. "재감지" 버튼을 눌러 다시 감지를 시도할 수도 있습니다.

한 번에 몇 장까지 처리할 수 있나요?

특별한 제한은 없지만, 브라우저의 메모리에 따라 처리 가능한 이미지 수가 달라질 수 있습니다. 일반적으로 수십 장의 이미지를 한 번에 처리할 수 있습니다.

모바일에서도 사용할 수 있나요?

네, 모바일 브라우저에서도 사용 가능합니다. 반응형 디자인을 적용하여 다양한 화면 크기에서 동작하지만, 정밀한 조정 작업은 데스크톱 환경에서 더 편리합니다.

원형과 사각형 모드의 차이는 무엇인가요?

원형 모드는 페트리 접시처럼 둥근 형태를 감지하는 데 최적화되어 있으며, Hough 변환 알고리즘을 사용합니다. 사각형 모드는 젤 전기영동 사진처럼 직사각형 영역을 감지하는 데 사용되며, 색상 분석과 에지 감지를 조합한 알고리즘을 사용합니다.

기술 정보

FORLAB은 순수 HTML5, CSS3, JavaScript로 구축되었습니다. 외부 라이브러리 없이 Canvas API를 사용하여 이미지 처리를 수행합니다. 주요 알고리즘은 다음과 같습니다:

  • Hough 변환 - 원형 객체 감지에 사용되는 고전적인 컴퓨터 비전 알고리즘
  • Sobel 에지 감지 - 이미지의 경계선을 추출하는 그레디언트 기반 방법
  • Otsu 이진화 - 최적의 임계값을 자동으로 계산하는 이미지 분할 기법
  • 색상/채도 분석 - HSL 색상 공간을 활용한 영역 분류 방법
  • 모폴로지 연산 - 팽창과 침식을 사용한 노이즈 제거 및 영역 정제
